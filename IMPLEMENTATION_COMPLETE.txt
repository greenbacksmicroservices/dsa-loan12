## 🚀 How to Use

### Create New Agent
```
1. Login as Employee
2. Go to: /employee/my-agents/
3. Click: "Add New Agent" button
4. Fill form:
   - Name: (required)
   - Phone: (required)
   - Email: (required)
   - Password: (required)
   - Photo: (optional - shows preview)
   - Gender: (optional)
   - Address: (optional)
   - PIN: (optional)
   - State: (optional)
5. Click: "Create Agent"
6. Success! Agent appears in:
   - Your agents list
   - Admin agent page
   - Can login with created credentials
```

### View Loan Details
```
1. Go to: /employee/all-loans/
2. Click: [View] button for any loan
3. See full loan details
4. Click: [Back] to return to list
5. Or click: [Edit] to modify loan
```

### Filter Loans
```
1. Go to: /employee/all-loans/
2. Use Search: Type applicant name/ID
3. Use Status Filter: Select loan status
4. Click Reset: To clear filters
5. All filters work in real-time
```

## 🔍 API Endpoints

### Working Endpoints
```
GET  /api/employee/all-loans/
     └─ Returns: List of all employee's loans

GET  /api/employee/loan/{id}/
     └─ Returns: Full loan details

POST /api/employee/add-agent/
     └─ Input: FormData with all agent fields
     └─ Returns: Success/error response

GET  /api/employee/my-agents/
     └─ Returns: List of employee's agents

POST /api/employee/upload-profile-photo/
     └─ Input: File upload
     └─ Returns: Success/error response
```
