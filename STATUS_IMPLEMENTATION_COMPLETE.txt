## 🚀 QUICK START (5 STEPS)

1. Run migrations:
   python manage.py makemigrations && python manage.py migrate

2. Update sidebar menu (add 2 links to base.html):
   /admin/new-entries/ and /admin/loan-details/

3. Test in browser:
   http://localhost:8000/admin/dashboard/

4. Start Celery (optional, for 24-hr automation):
   celery -A dsa_loan_management worker -l info
   celery -A dsa_loan_management beat -l info

5. Test workflow:
   Submit app → View in New Entries → Assign to Employee → Watch status change
